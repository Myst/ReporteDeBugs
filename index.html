Normas para reportar errores de juego

1.- Los errores deben estar explicados y sustentados por fuentes como : wowhead o wowfreakz. .

2.- Abstenerce a reportar los errores que anteriormente ya fueron publicados por otros usuarios.

3.- Un reporte para cada error.

4.- Se debe poner la etiqueta para el tipo de error de juego que esten reportando.

5.- No se admite reporte de fallos personales con los PJ como "mi pj se ha atascado" o "no puedo completar esta quest". Si se admitirán del tipo "Spell no funciona", "quest no funciona", etc.

6.- Un mal uso del bugtracker podrá conllevar a la expulsión de la cuenta GitHub para realizar reportes en el BugTracker.

7.- Cualquier otro tipo de reporte deberá hacerse via foro (aún no implementado) o vía skype: a myst.infernalwow, los errores como atascamiento u otros menores deberán ser reportados vía ticket.
